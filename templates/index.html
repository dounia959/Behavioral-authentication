How It Works

This system uses behavioral biometrics to continuously authenticate users. It learns unique patterns in: keystroke timing (dwell time, flight time), typing speed, mouse movement (speed, acceleration), and clicking patterns.

An ensemble of two anomaly detection models (SVM + Isolation Forest) flags sessions that deviate significantly from the learned pattern. Both models must agree for a window to be classified as legitimate.
